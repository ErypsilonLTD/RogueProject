Thank you for downloading Rogue Project!

Steps to run:
1. Open Settings and follow the steps below
 Windows 10:
  Time & Language → Language & Region → Administrative language settings → Change system locale → Beta: Use Unicode UTF-8 for world-wide language support
  Settings → Privacy → For developers → Default terminal application → Windows Console Host
 Windows 11:
  Time & Language → Language & Region → Windows display language → Beta: Use Unicode UTF-8 for world-wide language support
  System → Advanced → Terminal → Windows Console Host

2. Download .Net from Google at this link >https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/sdk-10.0.102-windows-x64-installer<
3. Run Rogue Project.exe found in this folder!
4. Enjoy <3
